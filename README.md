# Habitverse – Parallel Universe Explorer

## Project Overview
Habitverse is a gamified habit tracking system developed using Object-Oriented Programming (C++). The project transforms traditional habit tracking into an immersive narrative-driven experience.

## Objective
To design and implement a habit tracking application using OOP principles and gamification techniques.

## Technologies Used
- C++
- OOP Concepts
- Command Line Interface
- UML Diagrams

## OOP Concepts Implemented
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Composition

## System Architecture
Core Classes:
- Entity
- Guardian
- VoidBorn
- Habit
- Achievement
- Universe
- User

## Features
- Positive & Negative Habit Tracking
- Streak Monitoring
- Achievement System
- Progress vs Darkness Tracking
- Narrative-based Feedback
- Weekly Reports
