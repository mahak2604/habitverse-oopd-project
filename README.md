# Habitverse – Parallel Universe Explorer 🌌

## 📌 Project Overview
Habitverse is a gamified habit tracking system developed using Object-Oriented Programming in C++. It transforms traditional habit tracking into an immersive narrative-driven experience.

## 🎯 Objective
To implement a habit tracking system using OOP principles and gamification techniques.

## 🛠 Technologies Used
- C++
- Object-Oriented Programming
- Command Line Interface
- UML Diagrams

## 🧠 OOP Concepts Implemented
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Composition
